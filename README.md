# Sujan Anandh — Cinematic Portfolio v2

A cinematic, scroll-driven personal portfolio built with vanilla JavaScript, Three.js WebGL, and zero frameworks.

## ✨ Features

- **WebGL Shader Background** — Bronze-to-sapphire liquid wave shader that reacts to scroll position
- **Scroll-driven Slides** — 7 fixed slides that transition based on scroll progress
- **Typewriter Effect** — Hero tagline cycles through roles with realistic typing animation
- **3D Flip Project Cards** — Hover to flip and reveal GitHub links
- **Spinning Gradient Ring** — Animated conic-gradient border around profile photo
- **Journey Timeline** — Animated vertical timeline of milestones
- **Contact Glow** — Breathing radial glow and pulsing availability indicator
- **Custom Cursor** — Double-ring cursor with smooth lag
- **Particle System** — 500 additive spark particles (bronze + sapphire)
- **Wireframe Icosahedrons** — 3 rotating 3D wireframes in the background

## 🛠️ Tech Stack

- Vanilla JavaScript (ES Modules)
- Three.js (WebGL, custom GLSL shaders)
- HTML5 / CSS3
- Google Fonts — Italiana, Outfit, DM Mono
- No frameworks, no build tools, no dependencies
